void create_adj_matrix ( Graph *  graph,
char *  file_name

Abre o arquivo .txt e constrói uma matriz de adjacência.

Cada linha e coluna representa um vértice do Grafo e os índices são correspondentes ao vetor 'vertices' inicializado por 'get_vertices'. Dessa maneira, se o vértice na posição 4 possui uma aresta para o vértice na posição 2, então há um número 1 na quarta linha e segunda coluna da matriz.

Parameters
graphReferência a uma variável do tipo Graph que terá a matriz de adjacência inicializada.
file_nameString com o path para o arquivo .txt.
